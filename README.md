# flytoday-automation
این اسکریپت با استفاده از Python و Selenium فرآیند جستجوی بلیط در سایت FlyToday.ir را اتومات می‌کند. کد تمام پاپ‌آپ‌ها را به‌صورت هوشمند می‌بندد، شهر مبدا و مقصد را به شکل رندوم انتخاب می‌کند، یک تاریخ تصادفی از تقویم انتخاب می‌کند و در نهایت موجود بودن یا نبودن بلیط را بررسی و نتیجه را چاپ می‌کند.
